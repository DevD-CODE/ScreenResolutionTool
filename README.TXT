Esta clase apoya al programador de turno a obtener la resolución de la pantalla del dispositivo en que se esté ejecutando la aplicación Java, además de eso
apoya para crear una interfaz con un tamaño basada en el porcentaje introducido por parametros, este porcentaje será en base a la resolución de la pantalla, por ejemplo:
1600x900, entonces será 1600*.75 = 1200 de ancho para la interfaz de usuario, aplicable para JFrame pero también se puede aplicar en distintos ámbitos.

Finalmente, también es capaz de centrar la interfaz sin importar el tamaño dado, esta clase es útil al momento de crear aplicaciones responsivas que se ejecuten en
muchos dispositivos con resoluciones de pantalla distintos.

Para mas información vea la clase ScreenResolutionTool.java
